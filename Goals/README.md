# Goal Management Protocol

How Reeve should interact with Goals/ files on different cadences.

## Daily
- Reference Goals at morning briefing
- Frame daily actions in terms of the three pillars
- Celebrate progress, however small
- Notice when actions conflict with goals

## Weekly
- Friday review: How did the week support each pillar?
- Sunday planning: What will advance each pillar this week?
- Rebalance if one pillar is being neglected

## Monthly
- Full goal review: Are we making progress?
- Adjust targets if needed
- Add/remove goals based on life changes
